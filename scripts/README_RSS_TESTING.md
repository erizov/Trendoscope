# RSS Sources Testing

## Быстрый старт

```powershell
cd trendoscope2
.\scripts\test_rss_sources.ps1
```

## Что проверяется

- ✅ Доступность каждого RSS-источника
- ✅ Корректность формата (XML/RSS)
- ✅ Редиректы (301/302)
- ✅ Ошибки (404, 403, и др.)

## Результаты

Скрипт создает файл `rss_test_results.json` с детальными результатами.

## Использование перед добавлением нового источника

1. Добавьте URL в `news_sources.py`
2. Запустите `test_rss_sources.ps1`
3. Проверьте, что новый источник в статусе **OK**
4. Если есть проблемы - исправьте URL или удалите источник
